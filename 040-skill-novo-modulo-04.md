@.agents/skills/config-new-module

Alterar a skill atual para criar o módulo dentro do projeto frontend. Não alterar a funcionalidade que está correta que é a criação do módulo dentro da pasta modules e dentro do projeto backend.

No projeto frontend criar a seguinte estrutura:

- Rota Privada do Módulo: @page.tsx
- Página Principal do Módulo: @reports.page.tsx
- Componente Principal do Módulo: @reports.component.tsx

O exemplo acima foi feito como modelo usando o módulo de reports, mas os caminhos devem seguir o nome do módulo criado.
